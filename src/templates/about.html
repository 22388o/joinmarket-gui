JoinMarket

Joinmarket is a software tool to mix bitcoins with peers in a decentralized fashion.

Web UI

This web-based user-interface is built upon the JoinMarket API to make it user-friendly for non power-users.

Disclaimer

Both Joinmarket and this web-interface is in experimental phase, so commit your bitcoin funds at your own risk.
